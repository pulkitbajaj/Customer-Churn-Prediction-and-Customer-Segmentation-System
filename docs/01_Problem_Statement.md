# Customer Churn Prediction

## Business Problem

Telecommunication companies lose customers due to competition,
pricing issues, poor service quality, and customer dissatisfaction.

Retaining existing customers is significantly cheaper than
acquiring new customers.

## Problem Statement

Predict whether a customer is likely to churn based on
demographic, service usage, and billing information.

## Machine Learning Formulation

Type: Binary Classification

Target Variable:
- Churn = Yes (1)
- Churn = No (0)

## Objectives

1. Predict customer churn.
2. Identify key churn drivers.
3. Segment customers using clustering.
4. Build an interactive prediction dashboard.

## Success Metrics

- Accuracy
- Precision
- Recall
- F1 Score

Priority Metric:
Recall
